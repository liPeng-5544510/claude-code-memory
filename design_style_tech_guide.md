## 设计风格分析

### 配色方案
| 元素 | 颜色 | 用途 |
|------|------|------|
| 基底色 | 科技灰（浅灰 #E8E8E8, 深灰 #4A4A4A） | 背景、结构 |
| 强调色 | 橙色（#FF6B35） | 模块接口、箭头、重点 |
| 文字色 | 黑色 #000000 | 标题、正文 |

### 布局结构
```
┌─────────────────────────────────────────────┐
│           【大标题】                          │
│         副标题 | 核心要点1 | 核心要点2        │
├──────────────────┬──────────────────────────┤
│   左侧示意图     │      右侧示意图            │
│   (硬件/实物)    │     (软件/模块)           │
│   + 标注文字     │     + 标注文字             │
├──────────────────┴──────────────────────────┤
│           底部标注/品牌标识                   │
└─────────────────────────────────────────────┘
```

### 视觉元素特点
- **等距投影 (Isometric)**：3D 立体效果，2.5D 视角
- **线条简洁**：无冗余装饰，突出技术感
- **模块化设计**：组件清晰分离，体现工业级
- **标注风格**：技术标注 + 引导线

### 图形风格
- **示意图类型**：技术结构图、流程图
- **线条**：细线、轮廓清晰
- **填充**：渐变或纯色，低饱和度
- **细节**：DATA BUS、PROCESSING CORE 等技术标签

### 字体风格
- 标题：大号、粗体、黑色
- 副标题：中等字号、灰色
- 标注：小号、技术感

## 适用场景
技术白皮书、教程封面、架构图、技术文档封面
